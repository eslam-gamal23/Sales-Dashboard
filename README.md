# Sales Dashboard Project
This project presents a net sales dashboard created using Excel.

## Overview
The dashboard provides a detailed view of net sales performance across:
- Categories: Electronics, Furniture, Clothing
- Cities and Regions
- Age and Gender distribution
- Quantities Per Year

## Key Metrics
- Net Sales: 167,512,587
- Total Orders: 7,671
- Discounts: 49,576,701
- Net Profits: 32,159,852
- Profit Margin: 19.20%
- Profits per Order: 4,192
- Returns Rate: 4.01%
- Expense: 135,352,735
- Cancelles Rate: 4.35%

## Dashboard Highlights
- Sales by Year and Sub-Category
- Sales by City and Region
- Sales by Age-Category and Gender
- Category-wise Sales breakdown

## Tools Used
- Microsoft Excel
- Power Query
- Power Pivot
